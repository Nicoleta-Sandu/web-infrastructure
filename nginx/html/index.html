Web Infrastructure Demo

A containerized web application with monitoring

About This Demo

This demonstrates a complete web infrastructure setup with:

  • Nginx web server
  • Python backend API
  • PostgreSQL database
  • Prometheus and Grafana monitoring

API Endpoints

Test the following API endpoints:

  • /api/items - Get all items
  • /api/items/<id> - Get item by ID
  • /api/health - API health check
